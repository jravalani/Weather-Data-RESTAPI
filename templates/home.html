Particular Date Data

URL format: http://127.0.0.1:5000/api/v1/station/date

URL Example: http://127.0.0.1:5000/api/v1/10/1996-04-06

Particular Station Data

URL format: http://127.0.0.1:5000/api/v1/station

URL Example: http://127.0.0.0.1:5000/api/v1/10

Particular Station and Particular Year

URL format: http://127.0.0.1:5000/api/v1/station/year

URL Example: http://127.0.0.0.1:5000/api/v1/10/1995
